Reporting Problems with MacSweeper

No piece of software is perfect. If you have used any other computer program you have discovered this already. MacSweeper is no exception to this rule. If you have a problem with MacSweeper, you can get help.

First, make sure you are running the most recent version of MacSweeper. The version of MacSweeper can be determined by reading the About dialog under the File menu. You can download the most recent version of MacSweeper for free on the web.

If your problem is present in the most recent version of MacSweeper, please email the developer, Morgan Conbere, at mconbere@gmail.com.
