MacSweeper Controls

MacSweeper has been designed to replicate the feeling of Microsoft's Minesweeper as close as possible while maintaining the look and feel of a Mac OS X application. Some of the controls could not be carried over perfectly due to differences in available controls.

The following table explains the controls in MacSweeper.

Action

Description

Primary Mouse Click

Reveal Cell Action

The Primary Mouse Button is used to reveal cells on the game board. If you click on an empty or questioned cell, the cell will become revealed. If you click on a revealed cell and the cell is surrounded by the correct number of flags, the surrounding empty cells will be revealed.

Secondary Mouse Click

Flag Cell Action

The Seconday Mouse Button is used to flag cells. By clicking on a cell repeatedly the state will cycle between the available states. By default these are empty, flagged, and questioned, but the availability of the questioned state can be changed in the Game Style menu.

Alternative: ctrl-Primary Mouse Click
